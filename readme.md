# WellCorrelator

> 一款基于 PyQt6 的双井测井曲线对比分析工具，专为石油地质工作者设计。

![Python](https://img.shields.io/badge/Python-3.10%2B-blue)
![PyQt6](https://img.shields.io/badge/PyQt6-6.4%2B-green)
![Version](https://img.shields.io/badge/version-5.8-orange)

---

## 功能总览

- 支持 LAS / CSV / Excel 等多格式测井数据导入
- 双井并排显示，深度轴联动同步缩放
- 曲线充填（左/右/双向）可视化，阈值线可交互拖动
- 分层 (Top) 管理：添加、删除、重命名、批量录入、保存/加载 JSON
- Zone 自动生成：相邻 Top 之间的区间着色
- 层拉平：双击任意层位，两井在该层自动对齐
- Ghost 跨井对比：将一口井的曲线段拖入另一口井比较，支持上下位移 + 左右宽度拉伸
- DTW Zone 对比：计算两井同名 Zone 的动态时间规整距离与相似度百分比
- 联井连线叠加：共有层位之间自动绘制连接线
- 鼠标悬浮提示：显示当前深度及所在 Zone 厚度
- 深色专业 UI：A 井蓝色系 / B 井紫色系，侧边栏可折叠

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 2. 运行程序

```bash
python v5.8.py
```

### 3. 基本流程

1. 点击左侧 **「+ 加载测井文件」** 分别加载参考井 A 和目标井 B
2. 通过曲线下拉框切换要显示的测井曲线
3. 右键图区添加分层 Top，相邻 Top 自动生成 Zone 色块
4. 双击层位线触发层拉平，两井在该层垂向对齐
5. 右键 Zone 区域选择「DTW 对比」，弹窗显示两井相似度

## 版本迭代历史

### v1.0 — 曲线导入基础版

**核心目标：** 能把测井数据读进来并画出来。

**实现功能：**
- 支持 LAS 格式导入（依赖 `lasio` 库）
- 支持 CSV / TXT 文本格式导入（自动检测编码：UTF-8 / GBK / GB2312）
- 支持 Excel（`.xlsx` / `.xls`）格式导入
- 自动识别深度列（支持 `DEPTH` / `MD` / `深度` 等关键字）
- 曲线列名自动解析，去除特殊字符
- 单井曲线绘图（pyqtgraph PlotWidget，深度轴朝下）

---

### v2.0 — 双井并排 + 缩放对齐

**核心目标：** 把两口井放在一起看，联动缩放。

**实现功能：**
- 双井并排布局（QSplitter，可自由调整比例）
- 深度轴自动同步：拖动/缩放一口井，另一口井等比例跟随
- 曲线下拉框切换（加载后自动检测所有可用曲线名）
- 自动填充深度范围输入框（加载后即显示井的 Min/Max 深度）
- 深度范围手动输入 + 应用 + 重置
- 曲线值域范围手动输入 + 应用 + 重置
- 范围锁定功能（锁定后不可超出边界缩放）

---

### v3.0 — 曲线充填

**核心目标：** 让曲线不只是一条线，而是有填充颜色，更像专业测井图。

**实现功能：**
- 充填模式选择：无 / 左充填 / 右充填 / 双向充填（默认双向）
- 充填参考线（阈值线）可用鼠标拖动，实时更新充填区域
- 左色 / 右色独立调色（QColorDialog，支持透明度调节）
- 颜色按钮实时预览当前所选颜色
- 充填路径使用 QGraphicsPathItem 实现，性能优化

---

### v4.0 — 分层管理（Top & Zone）

**核心目标：** 加入地质分层功能，这是井对比的核心。

**实现功能：**
- 右键菜单：在图区任意深度处添加新 Top
- Top 属性：名称 + 深度（MD）+ 颜色
- TopColorManager：全局颜色管理，同名 Top 两井颜色统一
- Zone 自动生成：相邻两个 Top 之间自动构成一个 Zone，半透明背景着色
- ZoneColorManager：Zone 颜色同样全局统一
- Top 线可拖动修改深度（InfiniteLine movable=True）
- 右键菜单：重命名 Top / 删除 Top
- 分层保存为 JSON 文件 / 从 JSON 文件加载
- **批量添加 Top**：侧边栏文本框粘贴，每行 `层名 深度`，支持逗号或空格分隔
- 目标井选择（批量 Top 可添加到 A 或 B 任意一口井）

---

### v5.0 — Ghost 跨井对比

**核心目标：** 把一口井的某段曲线"幽灵化"叠加到另一口井上，做视觉对比。

**实现功能：**
- Ghost 选段模式：点击「GHOST 选段」按钮激活，十字光标，点击两次选定深度区间
- 选定区间的曲线以虚线彩色显示，拖入另一口井的绘图区
- **上下位移**：拖动 Ghost 顶部线，整体上下移动
- **锚点拉伸**：Ghost 内有分层 Top 作为锚点，可分段拉伸曲线形态
- **左右宽度拉伸**：左/右两条透明竖线控制 X 轴显示范围，模拟曲线值域缩放
- 右键 Ghost：弹出「删除 Ghost」选项
- 支持多个 Ghost 共存，每个 Ghost 自动分配不同颜色

---

### v5.5 — 层拉平 + 联井连线

**核心目标：** 实现专业软件的"层拉平"功能，两井按某个层位对齐浏览。

**实现功能：**
- **层拉平**：双击任意 Top 线（或右键菜单），两口井以该层为基准对齐深度轴
  - 再次双击同一 Top 取消拉平，恢复独立浏览
  - 状态栏显示偏移量：`A: xxxm  B: xxxm  偏移: ±xxm`
- **联井连线叠加（CorrelationOverlay）**：
  - 透明覆盖层，位于两口井中间
  - 自动检测两井共有的 Top 名称，用灰色虚线连接对应深度位置
  - 随缩放/平移实时更新连线位置
- 鼠标悬浮提示：显示当前深度值 + 所在 Zone 的厚度

### v5.8 — DTW Zone 相似度对比（当前版本）

**核心目标：** 给地质解释提供量化支持——用数字告诉你两口井的同一 Zone 有多像。

**新增功能：**
- **DTW 算法**（Dynamic Time Warping，动态时间规整）
  - 纯 numpy 实现，不依赖额外库
  - 计算两个时间序列的最小累计路径距离
  - 相似度归一化公式：`similarity = max(0, 100 × (1 - dist / max_possible))`
- **右键 Zone 触发对比**：在 Zone 背景区域右键，出现「📊 DTW 对比该 Zone: xxx」选项
- 自动定位另一口井中的**同名 Zone**
- 弹窗显示对比结果：
  - 两井 Zone 深度范围
  - 当前曲线名称
  - 有效数据点数（各自）
  - DTW 距离数值
  - 相似度百分比（0% ~ 100%）
- 信号路由：`dtw_compare_requested` 信号统一由主窗口 `_dtw_compare_zone()` 处理，不破坏原有数据模型

## 数据模型说明

```
WellData
├── name: str          # 井名（从文件读取或文件名）
├── df: DataFrame      # 测井数据（index = DEPTH）
└── topset: TopSet     # 分层集合
    ├── Tops: [Top]    # 按深度排序的层位列表
    │   ├── name: str
    │   ├── md: float  # 测深（m）
    │   └── color: str # 十六进制颜色
    └── Zones: [Zone]  # 由相邻 Top 自动构成的 Zone
        ├── name: "TopA → TopB"
        ├── md_from: float
        └── md_to: float
```

## 文件格式说明

### 输入格式

| 格式             | 说明                                       |
| ---------------- | ------------------------------------------ |
| `.las`           | 标准 LAS 2.0 测井文件，需安装 `lasio`      |
| `.csv` / `.txt`  | 纯文本，自动检测分隔符和编码               |
| `.xlsx` / `.xls` | Excel 表格，第一列或含关键词的列作为深度轴 |

### 分层保存格式（JSON）

```json
{
  "name": "WellA_Tops",
  "tops": [
    {"name": "T1", "md": 1200.5, "color": "#FF4444"},
    {"name": "T2", "md": 1350.0, "color": "#44FF44"}
  ]
}
```
